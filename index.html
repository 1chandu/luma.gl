LumaGL: A JavaScript WebGL Framework for Data Visualization

LumaGL's primary focus is to provide a set of efficient and easy-to-use WebGL building blocks enabling high-performance browser-based data visualizations.

Design Goals

  • Focus on data visualization. Prioritizes efficient rendering of large data sets of data and advanced visualization techniques.
  • Modular: set of small classes that can be used together with "raw" webgl code, or with components from other frameworks (e.g. stack.gl).
  • No "magic" global state. All components work with a standard WebGLRenderingContext. You don't need to initialize your context using luma.gl
  • Shader programming focus: built on top of glslify to facilitate shader programming
  • Headless support: and headless-gl for running WebGL under node
  • A modern, compact codebase that uses the latest JavaScript standards (ES6+)
  • Uses modern JavaScript build technologies (browserify/babel/npm etc).
